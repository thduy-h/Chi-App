# Development Guide

## Prerequisites
- Node.js 18+ (Node.js 20 recommended)
- npm

## Setup
1. Install dependencies:
```bash
npm install
```
2. Create env file:
```bash
cp .env.example .env.local
```

## Run in Development
```bash
npm run dev
```
Open `http://localhost:3000`.

## Build for Production
```bash
npm run build
```

## Run Production Build
```bash
npm run start
```
